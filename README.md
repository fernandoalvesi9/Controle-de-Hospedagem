# Controle de Hospedagem

Sistema de controle de hospedagem desenvolvido em Python com interface colorida no terminal.

## Descricao

Projeto Final desenvolvido por Fernando Alves.
O sistema permite registrar reservas de apartamentos, calcular diarias e gerar relatorios detalhados no console.

## Funcionalidades

- Cadastro de reservas com nome do funcionario, numero e tipo de apartamento
- Suporte a apartamentos Basic e Luxo
- Calculo automatico de diarias por pessoa e total geral
- Relatorio formatado com cores ANSI no terminal
- Validacao de entradas (nome, numero de pessoas, dias, tipo)
- Loop para registrar multiplas reservas na mesma sessao
